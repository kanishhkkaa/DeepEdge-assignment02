# DeepEdge ML Assignment
This project predicts the (x, y) coordinates of a bright pixel in a 50x50 image.
## Approach
- Generated dataset with 2500 images (all pixel positions)
- Used CNN model for prediction
- Split data into train, validation, and test sets
## Result
The model is able to predict coordinates with good accuracy.
## Tech Used
- Python
- NumPy
- TensorFlow / Keras
- Scikit-learn
